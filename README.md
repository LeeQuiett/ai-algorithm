## 개발 환경

- **코드 작성**: Windows에서 작성
- **컴파일 및 실행**: WSL (Windows Subsystem for Linux) 내의 리눅스 환경에서 실행
- **접속 방법**: SSH를 통해 Windows에서 리눅스 환경으로 접속하여 코드 컴파일과 실행을 진행
